Abstract

O sexto trabalho de caso era composto por criar um website relativo ao registo e atualização de canções.

O servidor foi desenvolvido utilizando a framework Express. Este teria de ser capaz de responder a pedidos de todos os tipos (GET,POST, DELETE e UPDATE) e estar devidamente preparado para o caso de receber como argumento algo "inesperado", evitando assim um cenário de erro por parte do servidor.

Graficamente o servidor é composto por uma página inicial que contém uma form de submissão de canções e abaixo uma tabela com todas as que foram criadas. Adicionalmente o utilizador dispões de duas opções para cada música, podendo assim apaga-la ou consulta-la. No caso da segunda opção este verifica todas as informações da música, podendo também edita-las.
