Abstract

O quarto trabalho de caso era composto por criar um website relativo aos arqueossítios.

Deste modo foi desenvolvida uma stylesheet que processava o ficheiro XML corresponde a todos os Arquessítios de Norte e gerava um ficheiro XML para cada um destes. Adicionalmente, estes foram agrupados por concelho de modo a ser possível uma melhor análise dos dados.

De seguida foi criada uma stylesheet que é associada a cada um desses ficheiros, de modo a serem convertidos em HTML (e minimamente visualmente estéticos).

Por fim, foi desenvolvido um servidor em node.js, acessível pela porta 7777 em localhost, que permite um acesso via browser a toda a informação processada.
