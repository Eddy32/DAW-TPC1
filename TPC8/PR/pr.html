Abstract

O oitavo trabalho de casa era composto por criar um Servidor para a visualização de todas as informações desejadas relativas aos Prémios Nobel.

O servidor desenvolvido utiliza MongoDB de modo a guardar toda a informação. A informação presente na base de dados é proveniente do ficheiro base prize.json. Adicionalmente foi desenvolvida uma api de dados capaz de responder a todos os pedidos GET desejados.
