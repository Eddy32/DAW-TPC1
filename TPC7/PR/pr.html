Abstract

O sétimo trabalho de caso era composto por criar um servidor NodeJS para a gestão de filmes., utilizando Mongo para armezanar os dados.

O servidor desenvolvido utiliza MongoDB para armazenar os dados. Os dados carregados para este são provenientes do ficheiro base movies.json

Graficamente o servidor é composto por uma página inicial que contém uma form de submissão de filmes e abaixo uma tabela com todas os presentes na base de dados. Adicionalmente o utilizador dispões de duas opções para cada filme, podendo assim apaga-la ou consulta-la. No caso da segunda opção este verifica todas as informações da música, podendo também edita-las.
